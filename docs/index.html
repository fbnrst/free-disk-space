What This Page Shows

This page displays detailed disk space usage analysis for GitHub Actions runners across different architectures and Ubuntu versions. The data is collected weekly using ncdu (NCurses Disk Usage) and helps you understand what's consuming disk space on GitHub runners.

Why is this important? Disk space limitations can cause CI/CD jobs to fail. By understanding which directories and files consume the most space, you can make informed decisions about what data to delete to free up space for your workflows.
